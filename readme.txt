# SSA app

## Django strategy
1. Dodaj apke do installed_apps (settings)
2. Models - jeżeli potrzebne
3. `python3 manage.py makemigrations <app-name>`
4. `python3 manage.py migrate <app-name>`
5. admin.py (jeżeli potrzebne) - dodaj model do CMS
6. formularze (jeżeli potrzebne)
7. views
8. templates (jeżeli są potrzebne)
9. local urls
10. global urls
11. test it
12. Dodanie widoku do Menu
